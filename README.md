## Description

Frontend repository made with React and [ant design](https://ant.design/).

## Requirements

- Node JS >= v14.17.4
- Run backend app (http://localhost:5000/api)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```
